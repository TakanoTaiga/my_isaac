# my_isaac
Offline isaac_ros2 container.

## How to use

Requiered
- Jetpack 5.1.2 or x86-64 Ubuntu 20.04
- NVIDIA CTK
- Ready to setup isaac ros

#### Build Docker container
```bash
git clone https://github.com/TakanoTaiga/my_isaac.git
cd ./my_isaac
./build.sh
```

#### How to run
```
cd ./my_isaac
run.sh
```
